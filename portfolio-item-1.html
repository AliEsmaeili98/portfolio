Library application

Manage your Books

Have you ever felt the need to manage all the books you have read or want to read? If yes this application is for you!

This application enables the user to store information about books including the name of the book, writer, publisher, translator and also his/her general opinion about the book.

Another feature is that users can search between all the books by any of the book's information or filter read ones and not read ones so he/she knows what books to buy next!

This application was built using Flutter and i learnt how to use some flutter packages including sqflite, speed dial, rating bar, native splash and launcher icon.
